# Algoritmos Estructurados — Proyectos Organizados

Esta es la reorganización de tus ejercicios y proyectos del curso. No se tocó
ninguna lógica ni funcionamiento de los programas: solo se movieron los
archivos a carpetas por tema y se agregó un pequeño comentario de título /
explicación en la parte de arriba de los ejercicios que no tenían uno.

## Estructura de carpetas

- **01-Variables y Operadores** — Ejercicios básicos de entrada de datos,
  operadores aritméticos y cálculos simples (área, edad, temperatura,
  nómina, calculadora, promedio, IMC, etc.)
- **02-Condicionales** — Ejercicios con `if / elif / else` (pares/impares,
  categorías de edad, préstamos, triángulos, años bisiestos, etc.)
- **03-Bucles While** — Ejercicios con el ciclo `while` (validaciones,
  contraseñas, cuenta bancaria, series numéricas, etc.)
- **04-Bucles For** — Ejercicios con el ciclo `for` (tablas de multiplicar,
  promedios, porcentajes, sumatorias, etc.)
- **05-Arrays** — Ejercicios con listas (rellenar, copiar, máximos, mínimos,
  sumas, promedios, etc.)
- **06-Diccionarios** — Ejercicios trabajando con diccionarios de Python.
- **07-Funciones** — Ejercicios y clases usando funciones (`def`), incluye
  la subcarpeta **Capturas TP-Funciones** con las capturas de pantalla del
  taller práctico de funciones.
- **08-Examenes y Parciales** — Ejercicios del Parcial 1.
- **09-Proyectos** — Los tres proyectos grandes del curso, cada uno en su
  propia subcarpeta:
  - **Proyecto Biblioteca**
  - **Proyecto Tienda Virtual** (incluye las distintas versiones que tenías:
    `Tienda.py`, `TiendaVirtual.py`, `TiendaVirtual2.0.py` y
    `proyecto_tienda_virtual.py`)
  - **Proyecto Gestion de Estudiantes** (incluye `Estudiantes.py` y
    `GestiondeEstudiantes.py`, que son dos versiones del mismo proyecto)
